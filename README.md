Signal Detection using Logistic Regression

This project explores how machine learning can detect signals in noisy environments.

Pipeline
1. Generate synthetic signal data
2. Add noise to simulate different signal-to-noise ratio (SNR) conditions
3. Train a logistic regression classifier
4. Study how noise affects classification performance

Files
- `generate_data.py` – generates synthetic signal data
- `data.csv` - contains teh synthetically generated data
- `logistic_regression.py` – trains and tests logistic regression on the generated data

Status
Work in progress. Future improvements include train/test split, feature extraction, and evaluation under varying SNR levels
